Como usar

Basicamente nosso aplicativo se basea em um controlador de sala, o qual é nescessário do uso do aplicativo para escanear o QR code onde utilizaremos com base o arduíno. Assim, enviando os dados ao banco de dados e registrando os tais.

Contudo, o professor poderá ter o acesso e controle para liberação da sala, onde será responsável por intermediar à caso ocorra algo.
